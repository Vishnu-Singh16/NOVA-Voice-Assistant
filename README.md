# 🎙️ NOVA Voice Assistant

## Overview
NOVA is an AI-powered voice-controlled Windows desktop assistant that executes system commands using natural language.

## Features
- Voice control with wake word ("OK NOVA")
- AI command routing (Groq LLaMA)
- File management (create, delete, rename)
- Smart folder navigation
- Application launcher
- Document finder
- Environment variable manager
- Shutdown / restart control
- GUI (CustomTkinter)

## Tech Stack
- Python
- Groq API
- SpeechRecognition
- Windows SAPI
- CustomTkinter

## Run
```bash
pip install -r requirements.txt
python gui.py
